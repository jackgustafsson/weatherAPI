# WeatherAPI App (OpenWeatherMap)

En enkel väderapplikation byggd med HTML, CSS och JavaScript. Appen hämtar väderdata från OpenWeatherMap och renderar resultatet dynamiskt i DOM:en baserat på användarens sökning.

## Funktioner

- Sök väder på stad via en sökruta
- Visar **aktuellt väder** (t.ex. temperatur, luftfuktighet, vind, soluppgång/solnedgång)
- Visar en **prognos** baserad på OpenWeatherMap:s 4-dagarsprognos i 3‑timmarsintervall (appens prognosvy väljer ut ett antal datapunkter och visar dem som separata block)
- Renderar om innehållet vid ny sökning (tidigare resultat rensas)
